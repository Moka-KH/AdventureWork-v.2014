# AdventureWork-v.2014

This project was developed as part of the Microsoft Power BI & SSAS Orientation Program held by EJADA Systems.
The focus of the program was to build and understand semantic models using SQL Server Analysis Services (SSAS) Tabular, and to visualize insights using Power BI.

Key Objectives
- Learn and apply semantic modeling concepts in SSAS Tabular.
- Design and deploy a tabular model with relationships, measures, and hierarchies.
- Connect Power BI Desktop to the SSAS model for live analysis.
- Build interactive dashboards and paginated reports using: Power BI Desktop

Tools & Technologies
- Microsoft SQL Server Analysis Services (SSAS Tabular)
- Microsoft Power BI Desktop
- DAX (Data Analysis Expressions)
- SQL Server Management Studio (SSMS)

Deliverables
- SSAS Tabular Model with business logic and DAX measures
- Power BI Interactive Dashboard connected live to the semantic model
